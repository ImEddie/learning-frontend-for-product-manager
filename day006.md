## 学习装 VS Code 插件

1. 点击 VS Code 左侧第五个 Extensitions，搜索 Prettier 并安装
2. `CMD + Shift + P -> Format Document`选择 Prettier 手动格式化代码
3. `Code -> Preferences -> Settings` 搜索 format，找到`format on save`, 打勾。（在 CMD + S 保存的时候自动格式化代码）
